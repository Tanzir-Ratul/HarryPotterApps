# Harry Potter Character App

This Android app is a simple example of how to build a Harry Potter character app using modern Android development tools and architecture, including Dependency Injection (Dagger Hilt), MVVM pattern, Retrofit, and Glide library.

## Features

- Display a list of Harry Potter characters with their images, names, and house names.
- Clicking on a character in the list navigates to a details page showing character information.

## Libraries and Technologies Used

- [Dagger Hilt](https://dagger.dev/hilt/) for dependency injection.
- [MVVM (Model-View-ViewModel)](https://developer.android.com/jetpack/guide) architecture pattern.
- [Retrofit](https://square.github.io/retrofit/) for making network requests.
- [Glide](https://github.com/bumptech/glide) for image loading and caching.
